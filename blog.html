var: The Scope of a var variable is Functional Scope. It can updated and re declared into the scope.
let: The scope of a let variable is block scope.It can be updated but cannot be re-declared into the scope.
const: The scope of a const variable is block scope. It cannot be updated or re-declared into the scope.

Undefined: Undefined means a variable has been declared but has yet not been assigned a value.
Null: Null is an assignment value. It can be assigned to a variable as a representation of no value.

REST API:A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style. For this reason, REST APIs are sometimes referred to RESTful APIs.
